# guess-the-number-name
# Привет! это моя первая курсовая работа,но уже вторая консольная игра. Эта игра называется "Угадай число" (guess the number)
Она написана на node.js
## описание функционала моей игры
компьютер загадывает любое число от 1 до 100,а игрок должен его угадать,пользуясь подсказками

** Возможности игры
* **подсказки:** игра дает подсказки после ввода числа `too high` - число больше или `too low` - число меньше загаданного
* **валидация ввода:** игра защищает игрока от некорректных ответов.Если ввести букву,пустую строку или число,которое не входит в диапазон,игра выводит предупреждение и попросит написать еще раз
* **учет времени и попыток:** игра фиксирует сколько секунд и сколько попыток потребовалось для победы
* **система рекордов:** результат игрока автоматически сохраняется в файл `records.json`. Таблица сортируется с приоритетом: сначала маленькое кол-во,а при их равенстве по лучшему времени.

## инструкция по запуску

### 1. откройте терминал, перейдите в папку с проектом и выполните установку зависимостей:
npm install
npm link

### 2. введите команду `guess` ,чтобы запустить игру


bin/guess.js - файл,который инициализирует окружение и запускает игру.
src/games/guess.js - модуль игры
records.json - база для хранения лучших результатов
pacage.json - файл проекта с описанием зависимостей
